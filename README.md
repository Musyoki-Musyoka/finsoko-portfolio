### Overview

Strategic AI Blueprint for Ethical SACCO Lending in East Africa. This portfolio presents FinSoko's three-agent ecosystem designed to fix the 68% loan denial rate for market vendors in Ujima SACCO by aligning lending with agricultural liquidity cycles and embedding ethics at every layer.

### Live Prototype

**[finsoko-portfolio-mw69.vercel.app](https://finsoko-portfolio-mw69.vercel.app/)**

The interactive web portfolio includes:

- **Executive Brief** — Board-ready strategic summary
- **Phase 1: Savannah Precision** — AIM, MAP, OCEAN frameworks
- **Phase 2: Tsavo Fluency** — 4D Framework, 3 Modalities, RAG architecture
- **Phase 3: Ethical Savannah** — ETHOS, TRACK, OASIS, PRIDE, HORIZON frameworks
- **Phase 4: Agent Savannah** — RANK, TRAIL, HUNT, GUARD, CYCLE frameworks
- **Live Prototype** — Interactive 3-scenario agent simulation (Grace, Amina, James)
- **Impact Assessment** — 5-year projections, financial inclusion metrics
- **Strategic AI Blueprint** — Diagnostic, Redesign, Impact, Reflection
- **Safari Reflection** — Personal learning journey across all four terrains

### Three-Agent Ecosystem

| Agent | Role | Framework |
|-------|------|-----------|
| **Scout** | Financial Literacy Coach | Detects stress signals, routes members |
| **Guardian** | Loan Triage | Validates income, scores risk, triggers Pause Points |
| **Hunter** | Human-in-Loop Coordinator | Briefs officers, enforces dignity filters |
